Chetan is a Python library that can be installed using pip. It is compatible with Python 3.8 and above. To install Chetan, you can use the following command:

```bash
pip install chetan
```
This will install the latest version of Chetan from the Python Package Index (PyPI).
You can also install Chetan from the source code by cloning the GitHub repository and running the setup script. Here are the steps to do that:
1. Clone the repository:

```bash
git clone https://github.com/snayu-ai/chetan
```

2. Navigate to the cloned directory:

```bash
cd chetan
```
3. Install the package using pip:

```bash
pip install .
```
This will install Chetan and its dependencies from the source code.
### Dependencies
Chetan has several dependencies that are required for its functionality. These dependencies are automatically installed when you install Chetan using pip. However, if you want to install them manually, you can find the list of dependencies in the `requirements.txt` file in the root directory of the repository.
### Optional Dependencies
Chetan has some optional dependencies that are not installed by default. These dependencies are required for specific features or functionalities. You can install them using the following command:

```bash
pip install chetan[all]
```
This will install all optional dependencies. You can also install specific optional dependencies by replacing `all` with the name of the specific dependency you want to install.
